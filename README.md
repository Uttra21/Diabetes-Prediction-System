# Diabetes Prediction System

## Overview
A machine learning-powered web application that predicts diabetes risk based on patient health parameters. The application provides user authentication, prediction history tracking, and an interactive analytics dashboard.

## Features
- User Registration & Login
- Diabetes Risk Prediction
- Prediction History Storage
- Dashboard Analytics
- Data Visualization
- Secure Authentication

## Tech Stack
- Python
- Flask
- SQLite
- Scikit-learn
- Pandas
- HTML/CSS
- JavaScript
