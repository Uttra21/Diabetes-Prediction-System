# Diabetes Prediction System

A Machine Learning-powered web application that predicts diabetes risk based on patient health parameters. The system provides risk probability estimation, prediction tracking, and an analytics dashboard for visualizing prediction insights.

---

## Features

* Diabetes Risk Prediction using Machine Learning
* Risk Probability Estimation
* Interactive Analytics Dashboard
* Prediction History Storage
* User Authentication System
* SQLite Database Integration
* Data Visualization and Reporting

## How It Works

1. User enters health-related information.
2. The trained Machine Learning model processes the input.
3. The system predicts whether the patient is likely diabetic or non-diabetic.
4. A risk probability score is generated.
5. Prediction details are stored in the database.
6. Dashboard analytics summarize historical predictions.

## Installation

### Clone the Repository

```bash
git clone https://github.com/Uttra21/Diabetes-Prediction-System.git
cd Diabetes-Prediction-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open in Browser

```text
http://127.0.0.1:5000
```

## Dataset

This project uses the Pima Indians Diabetes Dataset containing medical diagnostic measurements such as:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age
